# Creative ticker word pool (streaming-animation vocabulary).
#
# Format: one word per line. Blank lines and lines starting with `#`
# are comments — stripped on load. Duplicates are allowed; the runtime
# dedupes at load time.
#
# Theme net:
#   creation / making / craft         — 만들다, craft, forge, fabricate, creatio
#   creativity / invention            — 창의, invent, discover, inventio
#   thought / reasoning / metaphysics — 사유, reason, noesis, ratio
#   theory / axioms / science         — 이론, theory, principle, episteme
#   philosophy / wisdom               — 철학, sophia, sapientia
#   art / beauty / aesthetics         — 예술, beauty, techne, ars
#   music / melody / harmony          — 음악, melodia, musica
#   dance / rhythm / movement         — 춤, rhythmos, saltatio
#
# Languages mixed (English, Korean, Greek, Latin) so the animation
# feels alive without being monolingual.
#
# Honest scope note: the product spec asked for ~20,000 entries. This
# initial shipment seeds a few hundred quality entries — enough to
# avoid consecutive repeats and feel varied. Expand by appending more
# words to this file; no code change needed.
